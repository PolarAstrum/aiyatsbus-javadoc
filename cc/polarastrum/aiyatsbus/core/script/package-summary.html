Package cc.polarastrum.aiyatsbus.core.script

See: Description

  • Interface Summary 
    Interface Description
    AiyatsbusScriptHandlerAiyatsbus 脚本处理器接口负责管理不同类型的脚本处理器，如 Kether、JavaScript、Fluxon 等。 提供统一的脚本处理接口，支持多种脚本语言的执行。 根据脚本类型返回对应的处理器实例。
    ScriptHandler脚本处理器接口负责执行各种类型的脚本，如 Kether、JavaScript、Fluxon 等。 提供脚本的调用、预热和变量传递功能。 支持异步执行和批量脚本处理。
  • Enum Summary 
    Enum Description
    ScriptType脚本类型枚举定义了 Aiyatsbus 支持的脚本语言类型。 每种类型都有对应的脚本处理器实现。 支持多种脚本语言，满足不同的开发需求。
