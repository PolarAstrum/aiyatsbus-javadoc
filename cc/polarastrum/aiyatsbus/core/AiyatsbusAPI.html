Interface AiyatsbusAPI

  • All Implemented Interfaces:
    
public interface AiyatsbusAPI

    Aiyatsbus API 接口

    定义 Aiyatsbus 系统的核心 API 功能。 提供附魔管理、注册和语言系统的访问接口。 这是插件的主要入口点，所有核心功能都通过此接口访问。
