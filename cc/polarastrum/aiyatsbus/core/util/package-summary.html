Package cc.polarastrum.aiyatsbus.core.util

See: Description

  • Object Summary 
    Object Description
    BlockUtils方块工具类提供方块操作相关的工具函数，包括矿脉检测等。 使用广度优先搜索算法进行高效的方块探索。
    FileWatcher文件监听器对象提供文件变化监听功能，支持添加和移除监听器。 使用 TabooLib 的 FileWatcher 实现实时监听。 防止重复添加监听器和处理中的文件冲突。
    MathUtils数学工具对象提供表达式计算和权重选择功能。 使用 Crunch 库进行高性能的数学表达式计算。
    Reloadables重载管理器管理所有标记为可重载的方法和字段，提供统一的重载执行功能。 使用类访问器自动发现和注册可重载的组件。 支持配置文件的自动重载和方法的动态调用。
    VariableReaders变量读取器对象提供不同格式的变量读取器，支持大括号、双大括号和百分号格式。
    Vectors向量工具类提供向量操作相关的工具函数，包括安全速度设置、击退抗性处理等。 确保实体速度在安全范围内，防止客户端崩溃和作弊。
    YamlUpdaterYAML 配置文件更新器提供配置文件自动更新功能，支持增量更新和节点更新。 可以自动检测配置文件版本差异并更新到最新版本。 使用 Bukkit Configuration 加载配置，防止注释丢失。
  • Interface Summary 
    Interface Description
    Function1单参数函数接口接受一个参数但不返回值的函数接口。 适用于需要执行副作用操作的场景。
    Function2双参数函数接口接受一个参数并返回一个值的函数接口。 适用于需要转换或计算数据的场景。
    Function4To2四参数转双返回值函数接口接受四个参数并返回两个值的函数接口。 适用于需要同时返回多个结果的复杂计算场景。
    Function2To2双参数转双返回值函数接口接受两个参数并返回两个值的函数接口。 适用于需要同时返回多个结果的场景。
    VariableFunction变量函数接口定义变量替换的函数接口，支持返回多个替换值。
    SingleVariableFunction单变量函数接口定义单变量替换的函数接口，继承自 VariableFunction。
