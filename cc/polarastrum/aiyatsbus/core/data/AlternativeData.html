Class AlternativeData

  • All Implemented Interfaces:
    
public final class AlternativeData

    附魔额外数据类

    包含附魔的可选配置信息，这些数据一般不需要填写，使用默认值即可。 定义了附魔的各种行为属性，如是否可移除、权重、是否宝藏附魔等。
