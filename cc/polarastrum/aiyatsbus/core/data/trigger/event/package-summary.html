Package cc.polarastrum.aiyatsbus.core.data.trigger.event

See: Description

  • Class Summary 
    Class Description
    EventExecutor事件执行器类定义事件触发时的脚本执行逻辑，支持不同脚本类型和优先级设置。 负责在特定事件发生时执行相应的脚本代码。
    EventMapping事件映射类定义事件与附魔系统的映射关系，包含事件类、装备槽位、引用字段等配置。 用于将 Bukkit 事件映射到附魔触发系统中。
    EventResolver事件解析器类定义事件数据的解析逻辑，包含实体解析、事件解析和物品解析功能。 用于从 Bukkit 事件中提取相关的实体、事件和物品信息。
