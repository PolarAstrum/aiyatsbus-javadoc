Class Variables

  • All Implemented Interfaces:
    
public final class Variables

    附魔变量管理类

    负责管理附魔系统中的各种变量，包括变量的解析、计算和存储。 支持四种类型的变量：等级相关变量、物品相关变量、常量变量和自定义变量。 提供变量计算、修改和批量处理功能。
